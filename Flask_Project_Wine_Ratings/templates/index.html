What this is:

Use this API to convert your wine batch's chemical composition into a predicted wine rating! This exciting model has

Using the model:

Post a request to "https://wine-o-meter-43164.herokuapp.com/predict" in a .json format. The .json should contain a list of datapoints, and each datapoint should contain a list of the wine's attributes.

You should submit your data in the following order:
Fixed Acidity, Volatile Acidity, Citric Acid, Residual Sugar, Chlorides, Free Sulfur Dioxide, Total Sulfur Dioxide, Density, pH, Sulphates, Alcohol
